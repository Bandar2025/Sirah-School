# دليل مدير النظام - Al-Manara School ERP Admin Guide

## 1. التثبيت والتهيئة
* يتطلب النظام محرك Node.js وبيئة SQLite.
* يتم ضبط ملف `server.ts` للعمل عبر الشبكة المحلية (Server-Client mode).

## 2. إدارة المستخدمين والصلاحيات
* يمكن إنشاء حسابات متعددة (مدير، محاسب، مسجل، مدرس).
* يمتلك "المدير التقني" الصلاحية الكاملة لتعديل الجداول وتصفح قاعدة بيانات SQLite.

## 3. استراتيجية النسخ الاحتياطي
* **يومي**: تصدير ملف JSON.
* **أسبوعي**: نسخ ملف `data/school.db` يدوياً إلى سحابة خاصة أو فلاش خارجي.

## 4. استعادة البيانات
* في حال الانهيار:
  1. أعد تثبيت النظام.
  2. ضع ملف `school.db` القديم في مجلد `data`.
  3. سيفتح النظام كافة السجلات السابقة تلقائياً.

## 5. تحديث النظام
* عند صدور إصدار جديد، قم باستبدال ملفات الـ `dist/` فقط مع الحفاظ على مجلد `data`.
